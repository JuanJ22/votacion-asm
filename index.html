Configura Firebase para activar la votación en vivo

Sigue estos pasos — toma menos de 10 minutos y es completamente gratis.

1
Ve a console.firebase.google.com e inicia sesión con tu cuenta de Google. Crea un nuevo proyecto (cualquier nombre).
2
Dentro del proyecto: Build → Realtime Database → Create database. Elige la región más cercana y selecciona Start in test mode.
3
En la pestaña Rules de Realtime Database, reemplaza el contenido con esto y haz clic en Publish:
{ "rules": { ".read": true, "votes": { "$d": { "$vid": { ".write": "!data.exists()" } } } } }
4
Ve a ⚙️ Configuración del proyecto → General → Tus apps, agrega una app web (</>), copia el objeto firebaseConfig.
5
Abre este archivo HTML en un editor de texto, busca la sección FIREBASE_CONFIG y pega tus valores.
